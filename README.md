# JNoemi-ReparacionesAutomotrices-Sistemas
Es un sistema web de gestión para talleres mecánicos. Registra clientes, vehículos (placa/VIN) y emite órdenes de trabajo. Controla el proceso de reparación, almacena historiales clínicos automotrices en tiempo real y automatiza alertas por correo para el cliente. Optimiza tiempos mediante reportes.
